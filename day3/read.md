📌 Java printf() Formatting Practice

This program demonstrates how to use Java's printf() method for formatted output. It covers different format specifiers, width, precision, and flags to display data in a clean and readable format.

Concepts Practiced:
✔ printf() method
✔ Format specifiers (%s, %c, %d, %f, %b)
✔ Width formatting
✔ Precision formatting
✔ Formatting flags (+, -, 0, ,, (, space)
✔ Displaying different data types
✔ Output formatting
